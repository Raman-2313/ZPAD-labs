## Виконані візуалізації:
У роботі реалізовано наступні типи аналізу (згідно зі скріншотами):

1.  **Одновимірний аналіз:** Гістограми розподілу всіх ознак та детальний аналіз холестерину (Histogram + KDE Plot).
2.  **Аналіз діапазонів:** Кастомна гістограма пацієнтів за віковими групами (5 діапазонів).
3.  **Двовимірні залежності:** * Scatter plots для аналізу пульсу від віку.
    * Joint Plot з лінією регресії для оцінки артеріального тиску.
4.  **Кореляційний аналіз:** Теплова карта (Heatmap) всіх ознак датасету.
5.  **Порівняльний аналіз категорій:** Boxplot рівнів холестерину за різними типами болю в грудях.
6.  **Багатовимірна візуалізація:** * **4-D Bubble Plot:** Вік, пульс, холестерин та діагноз одночасно.
    * **3D Scatter Plot:** Просторова модель взаємозв'язку трьох показників.
    * **2D Density Plot (KDE):** Порівняння зон концентрації здорових та хворих пацієнтів.

## Інструкція із запуску

1. Перейдіть у директорію лабораторної роботи:
```bash
cd lab3
```

2. Створіть та активуйте віртуальне середовище:
```bash
python3 -m venv venv
source venv/bin/activate
```

3. Встановіть необхідні бібліотеки:
```bash
pip install -r requirements.txt
```

4. Запустіть Jupyter Notebook:
```bash
jupyter notebook
```

5. Відкрийте файл `lab3.ipynb` і виконуйте комірки послідовно. Скрипт автоматично перевіряє наявність даних на диску і використовує їх для побудови графіків.
