## Опис проєкту

У межах даної лабораторної роботи реалізовано програмний комплекс для моделювання та обробки гармонічних сигналів. Програма дозволяє генерувати функцію виду y(t) = A*sin(ωt +φ), накладати на неї стохастичний шум із заданими статистичними характеристиками та проводити цифрову фільтрацію результату в реальному часі.

### Ключові можливості:
- **Інтерактивне керування:** Регулювання амплітуди, частоти та фази через графічні елементи (слайдери).
- **Симуляція завад:** Накладання гауссового шуму з можливістю зміни середнього значення та дисперсії.
- **Цифрова обробка (DSP):** Фільтрація зашумленого сигналу за допомогою фільтра Баттерворта 3-го порядку (`scipy.signal.butter`).
- **Гнучка візуалізація:** Чекбокс для миттєвого перемикання між зашумленим та чистим сигналом для порівняльного аналізу.

## Інструкція із запуску

1. Перейдіть у директорію лабораторної роботи:
```bash
cd lab4
```

2. Створіть та активуйте віртуальне середовище:
```bash
python3 -m venv venv
source venv/bin/activate
```


3. Встановіть необхідні бібліотеки:
```bash
pip install -r requirements.txt
```

4. Запустіть Jupyter Notebook:
```bash
jupyter notebook
```

5. Відкрийте файл `lab4.ipynb` та виконайте код. Скрипт автоматично запустить готовий до роботи графік.
