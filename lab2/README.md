# Лабораторна робота №2

## Опис завдання
Метою роботи є опанування методів завантаження, очищення, підготовки та аналізу великих масивів даних (Big Data) за допомогою бібліотеки **Pandas**.

**Основні задачі:**
* **Частина 1 (VHI):** Робота з екологічними індексами, реалізація алгоритму переіндексації областей за українським алфавітом, фільтрація та вибірка даних.
* **Частина 2 (Power Consumption):** Обробка датасету, автоматизація збору даних, нормалізація/стандартизація та статистичний аналіз кореляцій.

## Системні вимоги
* **Операційна система:** Windows 10/11, macOS або Linux.
* **Інтерпретатор:** Python версії 3.10 або вище (рекомендовано 3.12).
* **Вільне місце на диску:** мінімум 200 МБ.

## Інструкція із запуску

### 1. Підготовка середовища
Відкрийте термінал (PowerShell або CMD) у папці з лабораторною роботою та встановіть необхідні залежності:

```bash
# Оновлення менеджера пакетів
python -m pip install --upgrade pip

# Встановлення необхідних бібліотек
pip install pandas numpy scipy notebook
```

### 2. Завантаження даних
Вам не потрібно завантажувати файли вручну. У ноутбуках реалізовано логіку автоматичної перевірки:

```bash
# Для VHI: дані зчитуються з локальної папки data/
# Для Power Consumption: код самостійно завантажує ZIP з UCI Repository
```

### 3. Запуск аналізу
Запустіть середовище Jupyter командою:

```bash
jupyter notebook
```

У браузері, що відкриється:
* Оберіть файл `lab2_1.ipynb` або `lab2_2.ipynb`.
* У верхньому меню натисніть **Cell** -> **Run All**.

## Технологічний стек
* **Pandas & NumPy** — обробка структурованих даних.
* **SciPy** — математичний апарат для розрахунку кореляції Спірмена.
* **Urllib & Zipfile** — автоматизація роботи з мережею та архівами.
* **Timeit** — прецизійне вимірювання часу виконання операцій.
