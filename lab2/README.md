# Лабораторна робота №2: Наука про дані

## Опис

Мета роботи — ознайомитися з методами завантаження, очищення та аналізу даних за допомогою бібліотеки **Pandas**.

**Частина 1:** робота з індексами VHI для областей України (завантаження, очищення, формування вибірок).
**Частина 2:** аналіз датасету електроспоживання (обробка даних, статистичний аналіз).

---

## Інструкція із запуску

### 1. Створення та активація віртуального середовища

Відкрийте термінал у папці `lab2`.

**Windows**

```bash
python -m venv venv
venv\Scripts\activate
```

**macOS / Linux**

```bash
python3 -m venv venv
source venv/bin/activate
```

---

### 2. Встановлення залежностей

```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```

---

### 3. Підготовка даних

* **VHI:** скрипт автоматично створює папку `data/` та завантажує туди файли з даними. Передбачено перевірку, щоб уникнути повторного завантаження.
* **Power Consumption:** код автоматично завантажує ZIP-архів з датасетом та розпаковує його.

---

### 4. Запуск

```bash
jupyter notebook
```

Відкрийте та виконайте ноутбуки:

* `lab2_1.ipynb`
* `lab2_2.ipynb`
