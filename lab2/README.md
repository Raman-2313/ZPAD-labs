# Лабораторна робота №2: Наука про дані

## Опис

**Мета:** опанування практичних навичок підготовки, обробки та аналізу даних у Python з використанням бібліотеки pandas. Робота передбачає завантаження та очищення структурованих даних VHI для областей України, заміну індексів на українську абетку, формування вибірок за роками та обчислення статистичних характеристик (min, max, середнє, медіана), а також роботу з великим датасетом електроспоживання: фільтрацію за критеріями, нормалізацію, стандартизацію, розрахунок кореляційних коефіцієнтів (коефіцієнт Пірсона та Спірмена), а також провести One Hot Encoding категоріального атрибута.

**Частина 1:** робота з індексами VHI для областей України (завантаження, очищення, формування вибірок).

**Частина 2:** аналіз датасету електроспоживання (обробка даних, статистичний аналіз).

---

* **Операційна система:** Windows 10/11, macOS або Linux.
* **Інтерпретатор:** Python версії 3.10 або вище.
* **Необхідні бібліотеки:** pandas, numpy, scipy, notebook.
* **Вільне місце на диску:** 200+ МБ (для зберігання завантажених даних).

## Інструкція із запуску

### 1. Створення та активація віртуального середовища

Відкрийте термінал у папці `lab2`.

**Windows**

```bash
python -m venv venv
venv\Scripts\activate
```

**macOS / Linux**

```bash
python3 -m venv venv
source venv/bin/activate
```

### 2. Встановлення залежностей

```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```
